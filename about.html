About This Project

This website is prepared as a class project for the course Internet Programming I during the 2024–2025 academic year at Unity University. The content in this site is primarily based on the book “የኢትዮጵያ ታሪክ” (Ye’Etyopya Tarik), which served as our main reference. Most of the written material is directly adapted from this book to preserve historical accuracy and detail.

The project was assigned by Ms. Gullat, and it aims to explore Ethiopian history, culture, events, and heritage using web technologies learned in class.

Project Description

This website project is a digital presentation titled “Ethiopian History”, designed to showcase Ethiopia’s rich historical and cultural heritage. The purpose of the site is to provide an engaging and informative platform for exploring key historical periods, ancient kingdoms, famous emperors, national events, and traditional festivals that shaped the country.

As part of the Internet Programming I course, the site was developed entirely using front-end technologies like HTML, ensuring accessibility and simplicity. It includes responsive layouts, structured content, internal linking between sections and pages, and visual highlights such as photos of emperors, festivals (like Timket, Meskel, and Irreecha), and landmarks (such as Lalibela and Axum).

Through this project, our team aimed to combine historical accuracy with modern web design practices to create an educational resource that reflects Ethiopia’s identity and pride.
